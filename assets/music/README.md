# วิธีเพิ่มเพลงในเครื่องเล่นเพลง

## ขั้นตอนการเพิ่มไฟล์เพลง

1. **เตรียมไฟล์เพลง**
   - รองรับไฟล์ MP3
   - แนะนำให้ใช้ชื่อไฟล์เป็นภาษาอังกฤษ เช่น `song1.mp3`, `song2.mp3`

2. **วางไฟล์เพลงในโฟลเดอร์**
   ```
   C:\xampp\htdocs\portfolio\assets\music\
   ```

3. **แก้ไขไฟล์ JavaScript**
   - เปิดไฟล์ `js/music-player.js`
   - แก้ไข array `playlist` ตามตัวอย่างด้านล่าง:

   ```javascript
   const playlist = [
       {
           name: 'ชื่อเพลงที่ 1',
           artist: 'ชื่อศิลปิน',
           file: 'assets/music/song1.mp3'
       },
       {
           name: 'ชื่อเพลงที่ 2',
           artist: 'ชื่อศิลปิน',
           file: 'assets/music/song2.mp3'
       }
   ];
   ```

4. **บันทึกและรีเฟรชหน้าเว็บ**

## ตัวอย่างโครงสร้างโฟลเดอร์

```
assets/
└── music/
    ├── song1.mp3
    ├── song2.mp3
    └── song3.mp3
```

## หมายเหตุ

- ไฟล์เพลงจะไม่ถูกรวมอยู่ในโปรเจกต์เนื่องจากขนาดไฟล์ใหญ่
- คุณต้องเพิ่มไฟล์เพลงของคุณเองในโฟลเดอร์ `assets/music/`
- ตรวจสอบให้แน่ใจว่าชื่อไฟล์ในโค้ดตรงกับชื่อไฟล์จริง
